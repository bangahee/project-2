# 기본 사칙연산 퀴즈

> Python 콘솔 기반 퀴즈 게임

- **언어**: Python 3
- **저장 방식**: JSON
- **실행 환경**: 터미널 / 명령 프롬프트

---

## 프로젝트 개요

Python을 활용하여 만든 콘솔 기반 퀴즈 게임입니다.  
사용자는 퀴즈를 풀고, 새로운 퀴즈를 추가하며, 저장된 퀴즈 목록과 최고 점수를 확인할 수 있습니다.  
JSON 파일(`state.json`)을 활용하여 프로그램 종료 후에도 퀴즈 데이터와 점수가 유지됩니다.  
Git을 사용하여 기능 단위로 커밋하고 버전을 관리했습니다.

---

## 퀴즈 주제 선정 이유

기본 사칙연산을 주제로 선택했습니다.  
누구나 쉽게 이해할 수 있는 주제이기 때문에, 퀴즈 내용보다는  
입력 처리, 예외 처리, 데이터 저장과 같은 핵심 기능 구현에 집중할 수 있다고 판단했습니다.

---

## 실행 방법

```bash
python3 main.py
```

프로그램 실행 후 메뉴에서 원하는 기능을 선택하여 사용할 수 있습니다.

---

## 기능 목록

### 1. 퀴즈 풀기
- 저장된 퀴즈를 순서대로 출제
- 정답 입력 후 정오답 판단
- 전체 결과 및 점수 출력

### 2. 퀴즈 추가
- 문제, 선택지 4개, 정답 번호 입력
- 입력값 검증 후 퀴즈 저장

### 3. 퀴즈 목록
- 저장된 전체 퀴즈 출력
- 총 퀴즈 개수 확인 가능

### 4. 점수 확인
- 최고 점수 확인
- 퀴즈 풀이 후 자동 갱신

### 5. 데이터 저장 (영속성)
- `state.json` 파일에 퀴즈와 점수 저장
- 프로그램 재실행 시 데이터 유지

### 6. 예외 처리
- 잘못된 입력 처리 (문자, 빈 입력, 범위 초과)
- `Ctrl+C` / `EOF` 입력 시 안전 종료
- 파일 없음 및 손상 시 자동 복구

## 데이터 파일 설명 (state.json)

퀴즈 데이터와 최고 점수를 JSON 형식으로 저장합니다.

```json
{
  "quizzes": [
    {
      "question": "문제",
      "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
      "answer": 1
    }
  ],
  "best_score": 3
}
```

**특징**
- UTF-8 인코딩 사용
- 파일이 없을 경우 기본 퀴즈 로드
- 파일이 손상된 경우 자동 복구

## Git 작업 방식

기능 단위로 커밋을 수행하고, 브랜치를 생성하여 기능 개발 후 병합했습니다.

### 브랜치 구조

| 브랜치명 | 설명 |
|---------|------|
| `main` | 최종 배포 브랜치 |
| `feature/quiz-play` | 퀴즈 풀기 기능 |
| `feature/add-quiz` | 퀴즈 추가 기능 |
| `feature/file-io` | 파일 저장/불러오기 기능 |
| `feature/exception` | 예외 처리 기능 |

### 사용한 Git 명령어

| 명령어 | 설명 |
|--------|------|
| `git init` | 로컬 저장소 생성 |
| `git add` | 변경 파일 스테이징 |
| `git commit` | 변경 사항 저장 |
| `git push` | 원격 저장소 업로드 |
| `git pull` | 원격 변경 사항 가져오기 |
| `git checkout` | 브랜치 이동 및 생성 |
| `git merge` | 브랜치 병합 |
| `git clone` | 저장소 복제 |
